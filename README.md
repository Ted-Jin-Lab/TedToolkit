# TedToolkit

You can check these packages on [Release](https://github.com/ArchiDog1998/TedToolkit/releases/latest).
The Visual Studio Version is `17.13.4`.

There are a lot of packages here.

| Package                                                             | Description                                           |
|---------------------------------------------------------------------|-------------------------------------------------------|
| [PureConst](src/libraries/TedToolkit.PureConst)                   | Make Pure Attribute works and add the Const Attribute |
| [Fluent](src/libraries/TedToolkit.Fluent)                         | Make your API as a Fluent API                         |
| [Assertions](src/libraries/TedToolkit.Assertions)                 | Fluent Assertions with three states                   |
| [InterpolatedParser](src/libraries/TedToolkit.InterpolatedParser) | Parse the data from string in a curse way             |
| [Roslyn Helper](src/libraries/TedToolkit.RoslynHelper)            | To help you with roslyn coding                        |
| [Grasshopper](src/libraries/TedToolkit.Grasshopper)               | Simplify your Grasshopper development                 |
